Projects & Research

Detecting Collusion in Multi-Agent Games

Simulated multiplayer games with colluding and honest agents using no-regret learning algorithms. Built a detection pipeline with XGBoost to identify colluders in real time using features from payoff and behavior deviations. Ongoing work focuses on auction environments and adversarial robustness.

Amber Lab: NLP Interfaces for Exoskeletons

Developing LLM-based user interfaces for lower-body exoskeletons at Caltech Amber Lab. Research focuses on interaction personalization and adaptive control (advised by Prof. Yisong Yue, Amy Li).

Bayesian Models of Electoral Behavior

Built an R package and generative model for simulating and analyzing congressional elections under uncertainty. Preparing library for publication (Profs. Katz & King).

Healthcare Cost Prediction (Hacktech Finalist)

Created a Streamlit app using XGBoost and CatBoost to predict insurance charges and generate model-informed counterfactuals. Visualized uncertainty in cost drivers.

CU Boulder Neuro Lab: Decision Modeling

Modeled Bayesian decision processes and compared performance against human data from Jones & Sieck (2003). Implemented in Java and visualized results.
